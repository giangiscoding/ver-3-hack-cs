### Top factors — SHAP-style (không cần SHAP thật)

Thay vì SHAP, tính **potential gain** cho mỗi tiêu chí:

```python
def compute_top_factors(layer_results) -> list[dict]:
    factors = []
    for layer in layer_results:
        for name, detail in layer.chi_tiet.items():
            gap = detail["max"] - detail["diem"]
            actual = detail["diem"]
            factors.append({
                "name": name,
                "diem_dat": actual,
                "diem_max": detail["max"],
                "gap": gap,           # điểm có thể cải thiện
                "gia_tri": detail["gia_tri"],
                "mo_ta": detail["mo_ta"],
            })
    # Top 3 yếu tố thiếu điểm nhiều nhất → đây là nơi DN cần cải thiện
    return sorted(factors, key=lambda x: x["gap"], reverse=True)[:3]
```

## Page 3 — Graph View

### Visualize bằng `pyvis` hoặc `streamlit-agraph`

```python
from pyvis.network import Network

def render_graph(mst, graph_data, fraud_report):
    net = Network(height="500px", directed=True)

    for node in graph_data["nodes"]:
        if node["id"] == mst:
            color = "#FF4B4B"   # đỏ — DN đang xem
        elif node.get("flag") == "shadow_controller":
            color = "#FFA500"   # cam — shadow controller
        elif node["id"] in fraud_report.cluster_members:
            color = "#FFD700"   # vàng — cùng fraud cluster
        else:
            color = "#90EE90"   # xanh — bình thường

        net.add_node(node["id"], label=node.get("ten", node["id"]), color=color)

    for edge in graph_data["edges"]:
        width = 3 if edge.get("flag") else 1
        color = "#FF0000" if edge.get("flag") else "#CCCCCC"
        net.add_edge(edge["source"], edge["target"],
                     title=edge["type"], width=width, color=color)

    return net
```

### Legend

| Màu | Ý nghĩa |
| --- | ------- |
| 🔴 Đỏ | DN đang được xem |
| 🟠 Cam | Shadow controller (node ẩn) |
| 🟡 Vàng | DN cùng fraud cluster |
| 🟢 Xanh | DN bình thường |
| Cạnh đỏ đậm | Quan hệ fraud |
| Cạnh xám | Quan hệ bình thường |
